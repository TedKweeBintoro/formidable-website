## Responsive Design

The website is fully responsive with breakpoints at:
- **Desktop**: 1200px+ (full layout)
- **Tablet**: 768px-1199px (adjusted grid layouts)
- **Mobile**: Below 768px (single column, mobile menu)

## Animations & Interactions
